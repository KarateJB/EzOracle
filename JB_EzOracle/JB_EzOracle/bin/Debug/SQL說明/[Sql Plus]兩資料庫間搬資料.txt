※兩資料庫間搬資料(適用於SQL-Plus)※

1.目的地還未建立表格
SET COPYCOMMIT 5000;
COPY FROM id/pwd@db_name TO id/pwd@db_name -
CREATE new_table_name -
USING SELECT * FROM exist_table_name;

2.目的地還已有此表格
SET COPYCOMMIT 5000;
COPY FROM id/pwd@db_name TO id/pwd@db_name -
APPEND new_table_name -
USING SELECT * FROM exist_table_name;
